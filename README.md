# 🧠 Mental Health Risk Prediction System

An AI-based web application that predicts mental health risk (Low, Medium, High) among teenagers based on their social media usage, lifestyle, and behavioral patterns.

---

## 📌 Project Overview

This project uses Machine Learning to analyze factors like:
- Social media usage 📱  
- Sleep patterns 🛌  
- Stress & anxiety levels 😰  
- Physical activity 🏃  
- Academic performance 📚  

👉 And predicts **depression risk level**

---

## 🚀 Features

- 🔮 Predicts Mental Health Risk (Low / Medium / High)
- 📊 Data Visualization (EDA)
- 🤖 Machine Learning Model (Random Forest)
- 🖥️ Streamlit Web App Interface
- 💡 Lifestyle Suggestions based on prediction
- ⚠️ Disclaimer for educational use

---
